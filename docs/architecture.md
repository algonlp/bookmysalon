# Architecture

- `src/api`: API routes, controllers, and middleware
- `src/config`: typed environment config
- `src/nlp/preprocessing`: text cleanup and tokenization
- `src/nlp/features`: feature extraction helpers
- `src/nlp/models`: train/evaluate/predict model stubs
- `src/nlp/pipelines`: orchestration for training workflow
- `src/shared`: logger and shared error types
